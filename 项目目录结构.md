## 关键目录说明

### 前端目录 (frontend/)
- **components/**: 可复用的React组件
- **pages/**: 页面级组件，按功能模块组织
- **services/**: API调用服务
- **store/**: 状态管理
- **types/**: TypeScript类型定义
- **utils/**: 工具函数

### 后端目录 (backend/)
- **controller/**: REST API控制器
- **service/**: 业务逻辑服务层
- **repository/**: 数据访问层
- **entity/**: JPA实体类
- **dto/**: 数据传输对象
- **config/**: 配置类
- **security/**: 安全相关类

### 数据库目录 (database/)
- **init/**: 数据库初始化脚本
- **migrations/**: 数据库版本迁移文件
- **seeds/**: 测试和开发用的种子数据

### 部署目录 (deployment/)
- **docker/**: Docker相关文件
- **aws/**: AWS云服务部署配置
- **nginx/**: Web服务器配置

## 开发环境要求

### 前端开发环境
- Node.js 18+
- npm 或 yarn
- TypeScript 5+

### 后端开发环境
- Java 17+
- Maven 3.8+
- MySQL 8.0+

### 数据库
- MySQL 8.0+
- Redis 6+ (可选，用于缓存)

### 开发工具
- IDE: IntelliJ IDEA
- 数据库管理: MySQL Workbench
- API测试: Postman
